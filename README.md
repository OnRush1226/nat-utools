## 内网穿透 NAT

基于 localtunnel 封装的 utools 内网穿透插件

欢迎 start ✨

![](https://cdn.jsdelivr.net/gh/lblblong/image-bed@main/1627284934657-QQ截图20210726144102.png)

## 安装

utools 中 搜索 `内网穿透NAT` ，或者下载 release 中的 upx 文件拖动到 utools 中安装
